1.Иницијализирај Git во директориумот: git init
2. Додај ги промените во контрола на верзии: git add .
3. Прати ги промените на GitHub: git push
4. Креирај нова гранка наречена "bug1": git checkout -b "bug1"
5. Провери на која гранка се наоѓаш: git branch
6. Префрли се на главната гранка: git checkout master
7. Креирај нова гранка наречена "bug2": git checkout -b "bug2"
8. Префрли се на "bug1" гранката, направи промени во "SILab1_1"
9. Префрли се на "bug2" гранката, направи промени во "SILab1_2"
10. Префрли се на "bug1" гранката, направи промени во "SILab1Main"
11. Префрли се на "bug2" гранката, направи промени во "SILab1Main"
12.Прати ги промените од "bug1" гранката во далечниот репозиториум:
13.Префрли се на главната гранка и преземи ги промените
14.Префрли се на "bug2" гранката и спои ги промените од главната гранка
15.Разреши ги конфликтите
16.Прати ги промените од "bug2" гранката во далечниот репозиториум
17.Спои со главната гранка во далечниот репозиториум
18.На крај, префрли се на главната гранка и преземи ги промените
